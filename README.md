# Wentworth Institute of Technology Workshops

## System Modeling, Control Design, and Hardware Integration with Simulink

This repository contains hands-on exercises for students, walking them through the end-to-end process of system modeling, control design, and hardware prototyping.

### Exercises:
1. Introduction to Simulink
2. Motor Modeling & Motor Control Design with Simulink
3. Mass-Spring-Damper System with Simscape
4. PID Control of a Mass-Spring-Damper
